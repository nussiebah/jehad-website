# 🦷 Dr. Jehad Ali Dental Clinic

A high-end, responsive landing page for a professional dental clinic. 

## ✨ Key Features
- **Modern UI:** Premium teal and gold aesthetic.
- **RTL Support:** Full support for Arabic typography and layout.
- **Interactive Booking:** Built-in appointment request form.
- **WhatsApp Integration:** Floating action button for direct patient contact.
- **Fully Responsive:** Works perfectly on mobile, tablet, and desktop.

## 🛠️ How to View
1. Download the `index.html` file.
2. Open it in any modern web browser (Chrome, Safari, Edge).

## 🚀 Deployment
This site is ready for **GitHub Pages**. Just push the files to a repository and enable "Pages" in the settings.
